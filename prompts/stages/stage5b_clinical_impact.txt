━━━ EXTRACTION INSTRUCTIONS ━━━

STEP 1 — CONDITION CLASSIFICATION
Map this article to ONE primary classification from the taxonomy above.
If the population spans multiple classifications, choose the most specific.
If no taxonomy entry matches, create a label and set taxonomy_match: false.

STEP 2 — OUTCOME MEASURE DISCOVERY
Scan the methodology and results sections. Identify EVERY validated outcome
measure used. For each measure:
- Extract the exact name and abbreviation as written by the authors
- Match to the MCID reference above if possible
- If not in the reference, use your training knowledge and flag as estimated

STEP 3 — ARM-BY-ARM DATA EXTRACTION
For each outcome measure, extract data for ALL study arms separately.
Do not average across arms. Report each arm's:
- Baseline mean (and SD if reported)
- Post-intervention mean (and SD if reported)
- Within-group change (delta)
- p-value for within-group change (if reported)

STEP 4 — BETWEEN-GROUP COMPARISONS
For each outcome measure, calculate or extract:
- Between-group difference at primary endpoint
- 95% confidence interval for between-group difference (if reported)
- p-value for between-group comparison
- Effect size (Cohen's d or Hedges' g) — calculate from means/SDs if not reported
- Whether the between-group difference meets, exceeds, or falls short of MCID

STEP 5 — WINNING ARM IDENTIFICATION
For each outcome measure, identify which arm achieved the greatest absolute
change from baseline. Report its protocol parameters precisely
(frequency, intensity, duration, number of sessions, technique specifics).
If no control/comparator exists, note "single arm — no comparison possible."

STEP 6 — CLINICAL LEVERAGE SCORE
Calculate a score from 0-10 using this rubric. Apply each criterion strictly:

+3  Between-group difference meets or exceeds MCID on primary outcome
+2  Effect size ≥ 0.8 (large) on primary outcome
+1  Primary outcome p < 0.05
+1  Winning arm protocol is fully specified (can be replicated: dose/frequency/duration all stated)
+1  Bias risk is low or moderate (not high or unclear)
+1  Study design is RCT or higher (Oxford 1a, 1b, or 2a)
+1  No spin detected

Sum these points. Maximum possible: 10.
